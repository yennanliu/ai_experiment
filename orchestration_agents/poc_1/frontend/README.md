# Notes App - Frontend

React + TypeScript note-taking application with search and tagging.

## Setup

```bash
npm install
npm run dev
```

App runs on http://localhost:5173

## Features

- Create, edit, delete notes
- Real-time search
- Tag-based organization
- Auto-save
- Responsive design
- Keyboard shortcuts

See `../workspace/frontend-spec.md` for full specification.
