## Concept

- What's LangChain
	- LangChain 是一個 Python 框架，用於開發由大語言模型（LLM）驅動的應用。它提供了一套標準接口和工具，使得開發者可以輕鬆構建 LLM 應用。

- What's LangGraph? 
	- LangGraph 是在 LangChain 基礎上構建的框架，用於構建可狀態追蹤、有向無環圖（DAG）結構的工作流。相比 Chain 的線性執行，LangGraph 提供了：
