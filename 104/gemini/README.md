# Auto apply 104 with gemini

- use `104_auto_apply_with_controls.js`


## Run

### Step 1 — Start Browser Session

Open a browser and navigate to:

https://www.104.com.tw

Log in manually and keep the browser session running.

---

### Step 2 — Run Auto Apply Script

Run:

`104_auto_apply_with_controls.js`

You can customize parameters such as:

- job keywords
- target locations
- number of applications
